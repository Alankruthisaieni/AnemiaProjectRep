# AnemiaProjectRep

**To run backend:**
Step-1: Redirect to the folder ` /server`
Step-2: To install dependencies: npm install
Step-3:  To run: npm start

**To run frontend:**
Step-1: Redirect to the folder ` /client`
Step-2: To install dependencies: npm install
Step-3:  To run: npm start
